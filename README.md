Project Overview
This project is a simple RESTful User Management API built to practice backend fundamentals such as endpoint design, input validation, and API testing.
The API allows clients to create and manage users for a hypothetical game application. It focuses on clean request/response handling, validation, and documentation using Swagger.
Tech Stack
Backend Framework: FastAPI
Language: Python
Validation: Pydantic
API Documentation: Swagger 
Testing: Swagger UI / Postman

Features
Create a new user
Retrieve all users
Retrieve a user by ID
Update user information
Delete a user
